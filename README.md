# Habit Tracker Application (Phase 3)

A Python-based habit tracking system using **Object-Oriented Programming (OOP)** and **Functional Programming** for data analysis.

## Features
- Modular architecture (Database, Logic, and UI are separated).
- Persistent storage with **SQLite**.
- Automated streak calculation for daily/weekly habits.
- Pre-filled with 4 weeks of test data for evaluation.

## To Start
1. Run `python setup_data.py` to populate the database with example data.
2. Run `python main.py` to start the application.
3. Run `pytest test_project.py` to verify functionality.
